6.4. Operation: Rollback

Die Anfrage ist ein Element vct:RollbackRequest, welches vom Typ vct:TransactionRequestType ist. Die Antwort ist ein Element vct:RollbackResponse, welches vom Typ vct:TransactionResponseType ist.

Regel: Rollback. Ein veloconnect-konformer Server muss folgenden Bedingungen genügen:

  1. Die Operation Rollback wird implementiert.
  2. Falls die durch dieAnfrage spezifizierte Transaktionsinstanz nicht existiert, wird der ResponseCode 420 zurückgeliefert.
  3. Falls die Anfrage eine Transaktionsinstanz identfiziert, die sich in einem der Zustände 50 bis 199 befindet, so ist der ResponseCode 430 zurückzuliefern.
  4. Ansonsten ist der ResponseCode 200 zurückzuliefern und die Transaktionsinstanz ist zu löschen.
