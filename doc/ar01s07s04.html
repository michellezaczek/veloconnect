7.4. Operation: ViewOrder

Die Anfrage ist ein Element vco:ViewOrderRequest, die Antwort ist ein Element vco:OrderResponse.

Regel: ViewOrder.  Ein veloconnect-konformer Server reagiert auf eine Anfrage ViewOrderRequest wie folgt:

  1. Zur Anfrage wird ermittelt, ob zu diesem Transaktionskontext eine Transaktionsinstanz existiert. Falls keine existiert wird der ResponseCode 420 zurückgeliefert.
  2. Aus dem Transaktionskontext werden die Elemente OrderResponseLine in die Antwort eingefügt. Die Antwort wird mit ResponseCode 200 an den Client ausgeliefert. Der Zustand der Transaktionsinstanz bleibt unverändert.
