15.2. Änderungen an Operationen/Transaktionen

  • GetProfile: Der Namen des Lieferanten (im Klartext) wird im Element vcp:SellersName mitgeteilt, vcp:Message kann zur Übermittlung einer Nachricht an den Nutzer des Client verwendet werden.
  • GetItemDetails: Wenn eine Bestellnummer unbekannt ist, wird dies explizit über ein vco:ItemUnknown-Element mitgeteilt und nicht wie in Version 1.0 durch das Fehlen eines Elements.
